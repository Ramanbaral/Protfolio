Jumping Japak Doremon

Built with :- HTML, CSS, JS

This is a very simple game built with HTML, CSS and JS. The idea for this game just randomly hit my mind when i was following my morning routine. In this game there is a character called doremon and doracake is his enemy. If he touches doracake then he is dead. The doracake keeps moving and he can jump around and be safe from doracake. This is a cartoon show called doremon and i watched it alot in my childhood. The olny difference between show and the game is the doracake. In show he loves eating doracake but in this game doracake is his enemy. This is a fun project and i enjoyed alot while building it and playing after building.
